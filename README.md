# cis-hardening
Hardens All Linux based operating systems

## Steps

1. Clone the reppository <br />
      `git clone https://github.com/kumarahul98/cis-hardening.git` <br />
3. Cd into the repository <br />
      `cd cis-hardening` <br />
5. Run the script <br />
      `bash main.sh` <br />
